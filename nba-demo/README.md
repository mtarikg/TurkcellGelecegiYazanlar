## NBA Demo
NBA Demo is a CRUD MVC application with a RESTful API, built as the final project of the .NET software development program.

### Project Walkthroughs
- [MVC Application Walkthrough](https://youtu.be/1wOcWH_riZM):
Visual and technical overview of the MVC application, including features, architecture, and functionality.
 
- [RESTful API Walkthrough](https://youtu.be/RR51d3ukwgM):
Overview of the API design, endpoints, and integration with the MVC application.

### Features
- Create, read, update, and delete NBA-related data
- MVC architecture with layered design and repository pattern
- RESTful API exposing application data
