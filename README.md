## .NET Software Development Program
### Overview
This repository contains projects completed as part of a .NET software development program, progressing from core language fundamentals to a full MVC application with a RESTful API.

#### 1. Basics
- Exercises focused on fundamental C# language features and programming logic

#### 2. SOLID Principles
- A simple scenario demonstrating the first two SOLID principles through code
- Written explanations with real-life examples

#### 3. Entity Framework
- Introduction to ORM concepts using Entity Framework

#### 4. MVC
- Introduction to the MVC pattern and its core concepts
- A small example application demonstrating MVC structure in practice

#### 5. Dependency Injection
- A simple example showcasing different types of dependency injection

#### 6. Final Project: [nba-demo](https://github.com/mtarikg/TurkcellGelecegiYazanlar/tree/main/nba-demo)
- A CRUD MVC application with a RESTful API based on NBA-related data
- Combines concepts such as MVC, dependency injection, and data access

### Tools
- **Languages & Frameworks:** C#, .NET
- **Data & Mapping:** Entity Framework, AutoMapper, SQL Server
- **Background Jobs:** Hangfire
- **UI:** Razor Pages

### Concepts
- **Design & Architecture:** SOLID Principles, Layered Architecture, Repository Pattern
- **Application Patterns:** MVC, Dependency Injection
- **Web & Security:** RESTful API, JWT, Sessions, Response Caching, Middleware
- **Cross-Cutting Concerns:** Logging
